# RouteX Demo Script

## Demo Order
1. Start the application from com.routex.Main.
2. Log in as System Admin.
3. Show UC11 by listing users and creating or updating one account.
4. Log in as Inventory Manager.
5. Show UC02 by listing inventory, then add or update an item.
6. Show UC04 by generating a shipment order.
7. Show UC05 by approving or rejecting an order.
8. Log in as Fleet Dispatcher.
9. Show UC06 by assigning a vehicle to an approved order.
10. Log in as Field Driver.
11. Show UC09 by updating shipment delivery status.
12. Show UC10 by reporting a vehicle issue.

## Key Talking Points
1. The same database supports all roles.
2. Services enforce business rules before persistence.
3. DAOs isolate SQL Server access.
4. Audit logs capture sensitive actions.
5. The interface is one integrated system, not separate demos.
